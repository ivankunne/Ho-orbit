Brand Overview

H-Orbit is a cosmic, community-driven music universe for the Dutch scene. The "orbit" metaphor runs through everything: a deep space-violet canvas, an orbital-ring logo, and a bento grid of glowing category cards that read like planets in a system the user navigates.

Personality

Modern, energetic, scene-native and slightly underground — closer to a creative collective or a music magazine than a corporate streaming service. It speaks to artists, bands and fans as insiders.

Tone of voice

Dutch, informal, direct, second-person (je / jouw). Short, verb-led labels paired with a one-line promise. Inviting and participatory — never clinical, never corporate.

Key design characteristics

  • Dark cosmic canvas with three ambient radial glows
  • Bento grid of genre-coloured category cards
  • A single violet action colour; everything else is structural neutral
  • Colour as taxonomy — 17 genre hues code content categories
  • Glassy low-contrast surfaces, hairline borders, glow shadows

Visual identity summary

A dark-violet, glow-lit, grid-first interface that feels like an interactive constellation. Bold white geometric headings, a single electric-violet CTA, thin-line uniform icons, restrained motion. The impression: premium-indie, nocturnal, music-forward.

09

Imagery & Visual Style

Iconography

Thin-line, uniform-stroke (Lucide-style). Monochrome, tinted to each card's genre colour, set in a 12px rounded tile.

Illustration

Abstract cosmic / orbital motifs — concentric rings, faint dotted constellation graphics. Geometric, low-opacity, decorative.

Image treatment

Dark, low-key. Genre-coloured 145° gradient washes over a near-black floor. Heavy translucency and backdrop blur (glassmorphism-lite).

Graphic elements

Glowing radial ambient fields, hairline white/10 borders, coloured glow shadows, pill / dot indicators in genre colours.

Decorative pattern

Orbital rings and dotted node motifs. The "orbit" metaphor is the unifying decorative language.

Logo

Clustered overlapping orbital-ring mark in a violet gradient with a violet halo, paired with the lowercase wordmark h-orbit.

Developer Rules

Typography

  • One typeface only — Space Grotesk, always with the fallback stack.
  • Headings: 700, pure white, tracking -0.025em.
  • Body 400 #ede8f5; secondary copy slate-400.
  • Use scale steps only — never invent sizes.

Colour

  • Violet #7c3aed is the only action colour — one primary CTA per view.
  • Structure with the orbit ramp; avoid arbitrary greys.
  • Use canonical #1a1528 background.
  • Genre colour is taxonomy — match icon, border, gradient, dot.
  • Always white text on violet / genre fills.

Spacing

  • 4px base — scale steps only (2px for fine tuning).
  • Card padding p-5 (20px); gutters 16–24px.
  • Compose with gap / space-*, not manual margins.
  • Reserve --bottom-nav-h on mobile.

Component consistency

  • Radii: controls 12px, cards 16px, pills full.
  • Use defined button variants; visible violet focus ring on all.
  • Cards follow the genre-gradient recipe exactly.
  • Build on Radix primitives — don't hand-roll a11y.
  • Motion restrained: 0.2–0.3s ease-out, subtle lift + glow.

Accessibility

  • Audit slate-400 #94a3b8 on dark surfaces — near the AA threshold for small text. Reserve for ≥14px / non-critical copy; step up to #ede8f5 for anything essential.
  • Verify each genre -400 text colour against its card floor for body-sized text; prefer them for labels/icons.
  • Never remove the violet focus ring (2px + offset); keep ≥44×44px tap targets in the bottom nav.
  • Respect prefers-reduced-motion; keep lang="nl" and real heading hierarchy; label icon-only buttons.
  • Disabled at 0.5 opacity can fail contrast — pair with cursor:not-allowed + supporting text.
